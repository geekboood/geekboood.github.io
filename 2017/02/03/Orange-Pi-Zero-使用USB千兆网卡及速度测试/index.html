Orange-Pi-Zero-使用USB千兆网卡及速度测试

Orange Pi Zero 是一个廉价的SBC，所以它只有一个百兆网口。于是我买来一个USB千兆网卡，并进行了速度测试。

简介

我所使用的是绿联的千兆USB3.0网卡。这款网卡使用了ASIX的AX88179方案，在台式机上面稍微测试了一下表现还不错。本文主要介绍该网卡在Linux下驱动的安装及速度的测试。
